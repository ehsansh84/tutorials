# ReadWriteOnce vs ReadOnlyMany vs ReadWriteMany

- ReadWriteMany: Multiple pods can write to the volume
- ReadWriteOnce: Only one pod can write to the volume
- ReadOnlyMany: Multiple pods can read to the volume
