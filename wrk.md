# How to do a load test using wrk?
### Step 1: Ù‘Install wrk:
#### Method 1:
```
sudo apt install wrk
```
#### Method 2: from source
```
git clone https://github.com/wg/wrk.git
cd wrk
make
./wrk -v
```
In case you do not have necessary tools, you may need to install:
```
apt install make
apt install unzip
apt install gcc
```
